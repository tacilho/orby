# Orby

Plataforma **open source** e **white-label** de chat em tempo real para organização de chamados entre equipes dentro de empresas.

A proposta é simples: entregar uma alternativa gratuita para times que precisam se comunicar e gerenciar atendimentos internos, mas não têm orçamento para ferramentas caras.

---

## Objetivo

Boa parte das soluções de service desk e comunicação corporativa é cara e distante da realidade de empresas pequenas ou em crescimento.  
O Orby existe para ser:

- gratuito  
- personalizável  
- simples de implantar  
- fácil de manter  

Sem exagero de features. Foco total em **resolver o problema**.

---

## Principais ideias

- Chat em tempo real  
- Organização de chamados por contexto ou equipe  
- Base preparada para customização de marca (white-label)  
- Código aberto para adaptação às regras de cada empresa  
- Estrutura que pode rodar localmente ou em nuvem

---

## Público-alvo

- pequenas e médias empresas  
- suporte interno  
- startups  
- operações que precisam cortar custo de licença

Se há verba para pagar plataformas grandes, provavelmente o Orby não é a escolha.

---

## Tecnologias

- Java 21  
- Spring Boot  
- JPA / Hibernate  
- JWT para autenticação  
- H2 em desenvolvimento  
- Lombok
